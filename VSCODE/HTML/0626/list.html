웹 표준 언어

html
웹 구조를 만들고 콘텐츠를 구성한다
DOM 구조인 트리 형태로 작성한다
css
웹 디자인을 담당하고, HTML에 CSS를 입힌다
선택자와 선언문 구조로 구성한다
선택자는 태그 선택자, 아이디 선택자, 클래스 선택자가 있다
javascript
웹 문서에서 움직임을 담당한다
객체 지향 언어로 움직임의 대상을 함수로 제어할 수 있다
