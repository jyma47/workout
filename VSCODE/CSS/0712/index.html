폰트 속성

폰트 크기는 font-size

폰트 글꼴은 font-family

이탤릭체 스타일 font-style

글꼴에 두께 주기 font-weight

폰트의 색상 color

문자 수평 정렬(왼쪽 가운데 오른쪽 양쪽) text-align

문자의 행간 (줄간격) line-height

문자 밑줄표시 text-decoration

자간 조정 letter-spacing

들여쓰기 text-indent

박스 안에서 문자를 가운데 정렬하기

박스에 문자를 입력하면 기본은 왼쪽 상단정렬됨 박스 가운데 정렬은 수평: text-align, 수직: line-height를 사용

웹 폰트 사용하기

  1. 구글 웹 폰트 사이트에 접속하기
  2. 사용하고 싶은 폰트를 찾아서 선택하기
  3. 문서에 폰트 연결코드 삽입하기 (1. html 2. css)
  4. css에서 font 속성 추가하기
