CSS 속성

가시 속성 (display)

display 속성은 보이고 숨기는 속성이다
개체를 숨기고 싶을 때 none 값을 사용
개체를 보일 때 블록 성격은 block 값
개체를 보일 때 인라인 성격은 inline 값

박스 속성 (width, height, border, margin, padding)

시각적으로 박스를 인식하는 것은 가로, 세로, 테두리, 배경색 속성이 결합된다 안쪽여백 padding은 박스에 추가되고, 바깥여백 margin은 박스 바깥으로 추가된다. 테두리와 배경색이 없으면 시각적으로 박스를 인식할 수 없다.
