CSS 선언방법 3가지

인라인 CSS : 태그에 직접 적용하기

내부 CSS : html 문서 head 안에 style 태그로 적용하기

외부 CSS : css 문서를 html에 link로 연결하여 적용하기

기본 선택자 3가지

태그 선택자 : 태그에 속성을 지정하는 것으로, 모든 태그에 적용됨

클래스 선택자 : 클래스는 점(.)으로 표시하고, 클래스 이름을 만들어 적용함

아이디 선택자 : 아이디는 샵(#)으로 표시하고, 아이디 이름을 만들어 적용함

아이디 선택자 : 레이아웃(영역)을 구성할 때 사용함

클래스 선택자 : 콘텐츠 디자인을 할 때 사용함
