1. I Love You

- 폴라로이드 사진 : 밖에 따기
- 사진 부분은 셰이프툴(라운드있는 네모)로 만들고 사진을 끌고와서 alt클릭
- 빨래집게 : 펜툴 따기
- 빨래줄 : 라운드 그리고 윗부분 날림 (줄 형태)
- 빨래 집게 색 바꾸기 (Hue / Saturation)
- 줄에다 마스크 씌워서 지워주면 걸린것처럼

2. 피자 레시피 만들기

(과제) 문자툴 실습 
- 문자 입력 (우리 걸어갈까) : 글자 (폰트패밀리 크기 조절 행간 조절 자간 조절 위쪽으로 옆쪽으로) / 래스터라이즈 타입 (이미지 편집) / 컨버트 투 셰이프 (패스 모양)
- 문단 입력 (시_삶이 그대를 속일지라도) : 시를 붙여넣고 가운데 정렬, 특정 강조 글자 꾸미기
- 패스 문자 (나비와 꽃) : 나비 패스 작업 / 텍스트 활성화 시켜서 패스 포인트에 가져다대고 클릭 / 글자 입력하고 꾸미기 / 다이렉트 셀렉션으로 패스 수정 / 꽃도 동일한 방식으로 진행
- Wrap Text : (파일 열기) 메리 크리스마스! 적기 / (T 와이어 그림) Create Warped Text / 모양에 따라서 작업 (Arc로 맞춰서) / Merry Christmas는 Arc를 - 값으로 맞춘다
