1. 장미
- 자석올가미로 영역을 둘러싸고 Add Layer Mask
- Shift 잡고 마스크판 누르면 배경이 보임
- 마스크 고리를 누르면 풀었다 걸었다 할 수 있다 > 내용물 수정? 마스크판 수정?
- 마스크판은 Black (덮어주는것) / White (빠지는것)
- 브러쉬 Size, Hardness, Opacity 로 칠해보기


2. grape
- 레이어 추가 후 네모 선 만들기
- Edit > Stroke / 선 OK 하고 Opacity를 50%
- Stroke 있는 레이어에 마스크 추가 / 마스크판 있는 쪽에서 작업
- Brush와 Eraser 는 반대 

3. 클리핑 마스크
- image 파일을 Alt 눌러서 클리핑 / 레이어 Transform
- 레이어 추가 후 브러쉬 하거나 Shape Tool 작업

-  원 (Feather 20px) 그리고 사진 가져와서 클리핑
- 기타 작업

4. BG + HappyBirthday
- 배경사진에 글자 레이어를 갖고 온다
- 레이어를 추가해 그라디언트를 깔고 클리핑 마스크
- 레이어를 추가해 흰색으로 눈 온 것처럼 칠한 후 클리핑 마스크
